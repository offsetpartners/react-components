IconText Example

Example:

```jsx
import { IconText } from "@offsetpartners/react-components";

<IconText icon="Mail" text="You've got mail!" />;
```

Done with HTML:

```html
<body>
  ...
  <div class="fig-icon-text"></div>
  ...

  <script>
    FigureReact = {
      IconText: {
        icon: "Mail",
        text: "You've got mail!",
      },
    };
  </script>

  <script src="https://unpkg.com/@offsetpartners/react-components@latest/dist/figure.js"></script>
</body>
```
